快速开始

1. 注册账号

访问 安装页面，输入邮箱获取验证码，30秒完成注册。

2. 安装 MCP

复制安装命令，粘贴给你的 AI 助手：

帮我安装这个 MCP 服务器：
{
  "syncagent": {
    "url": "https://api.syncagent.io/mcp",
    "headers": {
      "Authorization": "Bearer your_token_here"
    }
  }
}

3. 重启智能体

重启后，MCP Server 会自动连接并检测配置。

功能说明

配置备份

对 AI 说："备份我的配置"

系统会自动备份：

  • MCP 服务器配置
  • 自定义技能（Reasonix Skills）
  • 记忆文件（Memory）
  • 提示词规则（.cursorrules 等）

注意：API Keys 会自动脱敏，不会上传到云端。

配置恢复

在新设备上安装 MCP 后，对 AI 说："恢复我的配置"

系统会：

  • 自动检测当前智能体类型
  • 下载对应的配置
  • 写入到正确的路径
  • 提示需要手动填写的 API Keys

跨智能体迁移

从 Cursor 换到 Claude Desktop？

  1. 在 Cursor 中备份配置
  2. 在 Claude Desktop 中安装 MCP
  3. 系统自动检测到配置类型不同
  4. 提示"要从 Cursor 迁移吗？"
  5. 确认后自动转换格式并写入

常见问题

Q: API Keys 会上传到云端吗？

A: 不会。系统会自动检测并脱敏所有敏感信息（如 *_TOKEN、*_KEY），替换为占位符。恢复时会提示你需要手动填写。

Q: 支持多设备自动同步吗？

A: 当前版本需要手动备份/恢复。自动同步功能在开发中。

Q: 配置文件存储在哪里？

A: 存储在 Cloudflare R2（兼容 S3），数据加密传输和存储，仅你可以访问。

Q: 换智能体时，哪些功能可以迁移？

A: MCP 配置 100% 兼容。提示词和技能会尽量转换，不支持的功能会在迁移报告中说明。

Q: 免费版有限制吗？

A: 当前完全免费，无用户数和存储限制。未来可能推出付费版增加高级功能。

Q: 遇到问题怎么办？

A: 访问 GitHub Issues 提交问题，或发邮件到 support@syncagent.io
