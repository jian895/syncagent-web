2

复制下面的内容，粘贴给你的 AI 助手

帮我安装这个 MCP 服务器：

{
  "syncagent": {
    "url": "https://api.syncagent.io/mcp",
    "headers": {
      "Authorization": "Bearer 获取 Token 后自动填充"
    }
  }
}

提示：这条命令适用于所有智能体（Reasonix、Cursor、Claude Desktop 等）

3

重启智能体，开始使用

常用命令：

  • 备份我的配置 - 上传配置到云端
  • 恢复我的配置 - 从云端下载配置
  • 查看同步状态 - 查看云端备份情况

首次使用：

重启智能体后，MCP Server 会自动检测：

  • 如果云端没有配置 → 提示你"要备份当前配置吗？"
  • 如果云端有同类型智能体配置 → 提示你"要恢复吗？"
  • 如果云端有其他智能体配置 → 提示你"要迁移吗？"
