重启智能体，开始使用

常用命令：

  • 备份我的配置 — 把技能 / 记忆 / 人格上传到云端
  • 恢复我的配置 — 从云端写回本地（同智能体换设备）
  • 把云端 MCP 同步到本地 — 拉取你在网页托管的 MCP（含密钥）并写入本机
  • 查看同步状态 — 看备份与托管 MCP 概况

两套东西，分开管：

  • ✅ 养成资产（技能 / 记忆 / 人格）— 对话里备份 / 恢复
  • ✅ MCP 连接（含密钥）— 在 MCP 托管 网页填写，再说「同步到本地」

首次使用：

在智能体里说「备份我的配置」。AI 会按该客户端的清单收齐养成资产后上传。可随时到 我的备份 查看内容。
